## Setting up the Site
The repository contains an automated script to allow for re-creation of the website a user pushes anything to the repository.

1. Once the files/repoitory have been copied/or forked, ensure you rename the following folder - "RENAME" to ".github". 

2. Commit and Push all the files onced complete.

3. Accessing the GitHub Settings -> Pages tab select "gh-pages" under the branch tab, than press save.

4. Under the Settings -> Rules, create a rule for the main branch and selecting the following options
    - Restrict Deletion
    - Require a Pull Request Before Mergign
    - Block Force Pushes

This will ensure all students must create a branch to request merging of documents rather than merging to the main branch.
